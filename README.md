# ShopSphere Backend

A production-ready e-commerce backend built using Java and Spring Boot, designed to demonstrate modern backend engineering practices used in product-based companies.

## Planned Features

- User Authentication (JWT + RBAC)
- Product Catalog
- Shopping Cart
- Order Management
- Inventory Management
- Payment Processing (Mock)
- Notification Service
- Product Reviews
- Redis Caching
- Kafka Event-Driven Communication
- Docker Compose
- CI/CD with GitHub Actions
- Unit & Integration Testing

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Redis
- Apache Kafka
- Docker
- JUnit 5
- Mockito
- GitHub Actions

> 🚧 This project is being built incrementally to simulate real-world backend development.
